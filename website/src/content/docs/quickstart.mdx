## Get an API token

API tokens are created and managed in the ArcSite admin console. Sign in and open [the admin console](https://user.arcsiteapp.com#/admin) to generate a token, then use it as a Bearer credential on every request.

:::note
Keep your token secret and never commit it to source control. Replace `your_api_token_here` in the examples below with your actual token.
:::

## Know the base URL

All requests go to the following base URL:

`https://api.arcsite.com/v1`

## Make your first request

List the caller's projects to confirm everything is wired up correctly.

`GET https://api.arcsite.com/v1/projects`

<Tabs syncKey="lang">
<TabItem label="cURL">

```shell
curl "https://api.arcsite.com/v1/projects" \
  -H "Authorization: Bearer your_api_token_here"
```

</TabItem>
<TabItem label="JavaScript">

```js
const response = await fetch("https://api.arcsite.com/v1/projects", {
  headers: {
    Authorization: "Bearer your_api_token_here",
  },
});
const data = await response.json();
```

</TabItem>
<TabItem label="Python">

```python
import requests

response = requests.get(
    "https://api.arcsite.com/v1/projects",
    headers={"Authorization": "Bearer your_api_token_here"},
)
data = response.json()
```
