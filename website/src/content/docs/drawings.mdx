## Get Drawing

Returns drawing of your organization by project drawing id,

`GET https://api.arcsite.com/v1/drawings/<id>`

<Tabs syncKey="lang">
<TabItem label="cURL">

```shell
curl "https://api.arcsite.com/v1/drawings/<ID>" \
  -H "Authorization: Bearer your_api_token_here"
```

</TabItem>
<TabItem label="JavaScript">

```js
const response = await fetch("https://api.arcsite.com/v1/drawings/<ID>", {
  headers: {
    Authorization: "Bearer your_api_token_here",
  },
});
const data = await response.json();
```

</TabItem>
<TabItem label="Python">

```python
import requests

response = requests.get(
    "https://api.arcsite.com/v1/drawings/<ID>",
    headers={"Authorization": "Bearer your_api_token_here"},
)
data = response.json()
```

</TabItem>

### Query Parameters

| Parameter          | Default          | In    | Description                   |
| ------------------ | ---------------- | ----- | ----------------------------- |
| drawing_version_id | Optional[String] | query | The ID of the drawing version |

### Response Schema

| Name       | Type               | Description                |
| ---------- | ------------------ | -------------------------- |
| id         | String             | The ID of the drawing      |
| project_id | String             | The ID of the project      |
| name       | String             | The name of the drawing    |
| pdf_url    | String &#124; null | The URL of the PDF file    |
| png_url    | String &#124; null | The URL of the PNG file    |
| dxf_url    | String &#124; null | The URL of the DXF file    |
| tags       | List[String]       | Tags added to this drawing |

:::note
If the `drawing_version_id` is passed, the data of the specified version will be returned. If not, the data of the latest version will be returned by default.
:::

:::note
Please note that all URLs are not guaranteed and may be null if the app fails to upload to the cloud.

The returned urls will expire in 24 hours.
:::

:::note
The pdf could take some time(often less than 1 minute) to generate if the drawing contains location based photos, the `pdf_url` could be null before it has been generated.
:::

## Get Drawing Location Photos

Returns location photos associated with a drawing.

`GET https://api.arcsite.com/v1/drawings/<id>/location_photos`

<Tabs syncKey="lang">
<TabItem label="cURL">

```shell
curl "https://api.arcsite.com/v1/drawings/<ID>/location_photos" \
  -H "Authorization: Bearer your_api_token_here"
```

</TabItem>
<TabItem label="JavaScript">

```js
const response = await fetch("https://api.arcsite.com/v1/drawings/<ID>/location_photos", {
  headers: {
    Authorization: "Bearer your_api_token_here",
  },
});
const data = await response.json();
```

</TabItem>
<TabItem label="Python">

```python
import requests

response = requests.get(
    "https://api.arcsite.com/v1/drawings/<ID>/location_photos",
    headers={"Authorization": "Bearer your_api_token_here"},
)
data = response.json()
```

</TabItem>
</Tabs>

```json title="Response"
{
  "location_photos": [
    {
      "url": "https://cdn-files-1.arcsite.com/location-photo-1.png?Expires=1642318765&Signature=..."
    },
    {
      "url": "https://cdn-files-1.arcsite.com/location-photo-2.png?Expires=1642318765&Signature=..."
    }
  ]
}
```

### Query Parameters

| Parameter          | Default          | In    | Description                   |
| ------------------ | ---------------- | ----- | ----------------------------- |
| drawing_version_id | Optional[String] | query | The ID of the drawing version |

### Response Schema

| Name            | Type         | Description                         |
| --------------- | ------------ | ----------------------------------- |
| location_photos | List[Object] | List of location photo objects      |

### Location Photo

| Name | Type   | Description                          |
| ---- | ------ | ------------------------------------ |
| url  | String | The presigned URL of the photo file  |

:::note
If the `drawing_version_id` is passed, the location photos of the specified version will be returned. If not, the location photos of the latest version will be returned by default.
:::

:::note
The returned URLs will expire in 24 hours.
:::

## Get Drawing Payment

Returns the payment data for a drawing, including balance schedules and deposit amount.

`GET https://api.arcsite.com/v1/drawings/<id>/payment`

<Tabs syncKey="lang">
<TabItem label="cURL">

```shell
curl "https://api.arcsite.com/v1/drawings/<ID>/payment" \
  -H "Authorization: Bearer your_api_token_here"
```

</TabItem>
<TabItem label="JavaScript">

```js
const response = await fetch("https://api.arcsite.com/v1/drawings/<ID>/payment", {
  headers: {
    Authorization: "Bearer your_api_token_here",
  },
});
const data = await response.json();
```

</TabItem>
<TabItem label="Python">

```python
import requests

response = requests.get(
    "https://api.arcsite.com/v1/drawings/<ID>/payment",
    headers={"Authorization": "Bearer your_api_token_here"},
)
data = response.json()
```

</TabItem>
</Tabs>

```json title="Response"
{
  "balance": {
    "payment_schedules": [
      {
        "name": "Deposit",
        "amount": 500.00,
        "percentage": 25.00
      },
      {
        "name": "On Completion",
        "amount": 1500.00,
        "percentage": 75.00
      }
    ]
  },
  "deposit": 500.00
}
```

### Query Parameters

| Parameter          | Default          | In    | Description                   |
| ------------------ | ---------------- | ----- | ----------------------------- |
| drawing_version_id | Optional[String] | query | The ID of the drawing version |

### Response Schema

| Name    | Type   | Description                                    |
| ------- | ------ | ---------------------------------------------- |
| balance | Object | The balance information of the drawing payment |
| deposit | Number | The deposit amount                             |

### Balance

| Name      | Type         | Description                      |
| --------- | ------------ | -------------------------------- |
| payment_schedules | List[PaymentSchedule] | List of payment schedule objects |

### PaymentSchedule

| Name       | Type   | Description                                          |
| ---------- | ------ | ---------------------------------------------------- |
| name       | String | The name of the payment schedule                        |
| amount     | Number | The amount of the payment schedule                      |
| percentage | Number | The percentage of the payment schedule (0.00 - 100.00)  |

:::note
If the `drawing_version_id` is passed, the payment data of the specified version will be returned. If not, the payment data of the latest version will be returned by default.
:::
