Drawings

Get Drawing

Section titled “Get Drawing”

Returns drawing of your organization by project drawing id,

GET https://api.arcsite.com/v1/drawings/<id>

  • cURL
  • JavaScript
  • Python

Query Parameters

Section titled “Query Parameters”
ParameterDefaultInDescription
drawing_version_idOptional[String]queryThe ID of the drawing version

Response Schema

Section titled “Response Schema”
NameTypeDescription
idStringThe ID of the drawing
project_idStringThe ID of the project
nameStringThe name of the drawing
pdf_urlString | nullThe URL of the PDF file
png_urlString | nullThe URL of the PNG file
dxf_urlString | nullThe URL of the DXF file
tagsList[String]Tags added to this drawing

Note

If the drawing_version_id is passed, the data of the specified version will be returned. If not, the data of the latest version will be returned by default.

Note

Please note that all URLs are not guaranteed and may be null if the app fails to upload to the cloud.

The returned urls will expire in 24 hours.

Note

The pdf could take some time(often less than 1 minute) to generate if the drawing contains location based photos, the pdf_url could be null before it has been generated.

Get Drawing Location Photos

Section titled “Get Drawing Location Photos”

Returns location photos associated with a drawing.

GET https://api.arcsite.com/v1/drawings/<id>/location_photos

  • cURL
  • JavaScript
  • Python

Query Parameters

Section titled “Query Parameters”
ParameterDefaultInDescription
drawing_version_idOptional[String]queryThe ID of the drawing version

Response Schema

Section titled “Response Schema”
NameTypeDescription
location_photosList[Object]List of location photo objects

Location Photo

Section titled “Location Photo”
NameTypeDescription
urlStringThe presigned URL of the photo file

Note

If the drawing_version_id is passed, the location photos of the specified version will be returned. If not, the location photos of the latest version will be returned by default.

Note

The returned URLs will expire in 24 hours.

Get Drawing Payment

Section titled “Get Drawing Payment”

Returns the payment data for a drawing, including balance schedules and deposit amount.

GET https://api.arcsite.com/v1/drawings/<id>/payment

  • cURL
  • JavaScript
  • Python

Query Parameters

Section titled “Query Parameters”
ParameterDefaultInDescription
drawing_version_idOptional[String]queryThe ID of the drawing version

Response Schema

Section titled “Response Schema”
NameTypeDescription
balanceObjectThe balance information of the drawing payment
depositNumberThe deposit amount

Balance

Section titled “Balance”
NameTypeDescription
payment_schedulesList[PaymentSchedule]List of payment schedule objects

PaymentSchedule

Section titled “PaymentSchedule”
NameTypeDescription
nameStringThe name of the payment schedule
amountNumberThe amount of the payment schedule
percentageNumberThe percentage of the payment schedule (0.00 - 100.00)

Note

If the drawing_version_id is passed, the payment data of the specified version will be returned. If not, the payment data of the latest version will be returned by default.
